# Random Password Generator (GUI)

Графическое приложение на Python для генерации надежных случайных паролей с гибкой настройкой параметров и сохранением истории.

## Автор
* **Имя Фамилия:** Ярослава Лепехина

## Описание функционала
* **Настройка параметров:** выбор длины пароля от 4 до 32 символов с помощью удобного ползунка.
* **Кастомизация состава:** включение/выключение цифр, латинских букв и специальных символов.
* **Безопасность генерации:** использование встроенной библиотеки `random`.
* **Валидация:** проверка на граничные значения длины и обязательный выбор хотя бы одного набора символов.
* **История:** сохранение записей генерации в локальный файл `passwords_history.json` и автоматическая загрузка истории при новом запуске приложения.
* **Удобство:** быстрая кнопка копирования полученного пароля в буфер обмена системы.

## Инструкция по запуску

### Требования
* Установленный интерпретатор **Python 3.8 или выше**.
* Никаких сторонних библиотек устанавливать не нужно (используется встроенный пакет `tkinter`).

### Запуск приложения
1. Склонируйте репозиторий или скачайте архив с файлами.
2. Откройте терминал/консоль в папке проекта.
3. Запустите команду:
   ```bash
   python main.py
   ```

### Запуск тестов
Для проверки стабильности функций генерации и валидации запустите юнит-тесты:
```bash
python -m unittest test_password.py
```

## 🧪 Примеры использования (Тест-кейсы)

1. **Позитивный сценарий:** Выберите длину `16`, активируйте все чекбоксы, нажмите «Сгенерировать». В поле отобразится стойкий пароль, а в правой таблице появится новая строчка с текущим временем.
2. **Граничное значение:** Сдвиньте ползунок на минимум (`4`) или максимум (`32`). Приложение корректно сгенерирует пароль ровно заданной длины.
3. **Негативный сценарий:** Отключите все три чекбокса (цифры, буквы, спецсимволы) и нажмите кнопку генерации. Приложение выдаст предупреждающее окно: *«Выберите хотя бы один тип символов!»*, предотвратив сбой программы.
